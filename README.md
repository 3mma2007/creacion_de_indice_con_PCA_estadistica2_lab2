# Lab 2 - Estadística y Probabilidad II

Análisis multivariado sobre métricas de monitoreo de infraestructura distribuida (CPU, memoria, red, I/O, latencia).

## ¿Qué se hace?

- Muestra aleatoria de 1500 observaciones del dataset original
- Clustering, ACP no lineal y análisis factorial para explorar la estructura de los datos
- Regresión lineal múltiple usando los componentes obtenidos

## Archivos

- `script_r.R` — análisis completo en R
- `datos_lab2.xlsx` — dataset fuente
- `informe.pdf` — resultados y conclusiones

# Lab 2 - Estadística y Probabilidad II

Análisis multivariado sobre métricas de monitoreo de infraestructura distribuida (CPU, memoria, red, I/O, latencia).

## ¿Qué se hace?

- Muestra aleatoria de 1500 observaciones del dataset original
- Aplicar PCA (Principal Component Analysis).
- Crear un indice a partir del PCA.
